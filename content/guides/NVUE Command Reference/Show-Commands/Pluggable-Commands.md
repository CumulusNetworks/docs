## nv show interface \<interface-id\> pluggable

### Usage

`nv show interface <interface-id> pluggable [options]`

### Version History

Introduced in Cumulus Linux 5.3.0

### Example

```
cumulus@leaf04:mgmt:~$ show interface swp1 pluggable
```
